Ox Osanix / Ozalyn — Link-Building Guide

For the VA. Goal: publish the ready-made articles below on free Web-2.0 blogs, each linking to osanix.shop, to build a natural, varied backlink profile. Follow every rule exactly.

Golden rules — read first

One fresh IP + one fresh email + one fresh account per article. Never create two accounts on the same IP or the same email.
Match the account's IP country to the article's language (German article → German IP, Danish → Danish IP, Dutch → Dutch IP, English → UK IP).
Publish the article in the language it is written in. Never mix languages on one blog.
Insert only the links listed under each article, with the exact anchor text shown. Do not add extra links.
Space the work out — 2 to 3 articles per day, maximum. Do not publish everything at once.
Do not link the blogs to each other. Each blog stands alone.
After publishing, copy the live URL into the tracking sheet (Step 4).

1 Set your IP + browser

  1. Open a fresh browser profile (Chrome → profile menu → Add → new profile), or an incognito window with the proxy set. Use one profile per account.
  2. Turn on the residential proxy your manager sent you and set its country to the article's country: EN → United KingdomDE → GermanyDK → DenmarkNL → Netherlands
  3. Request a new IP for every new account (rotate the proxy session). Confirm the country at whatismyipaddress.com before signing up.
  4. Create a new free email for each account (Gmail or Outlook). One email is used for exactly one blog account.
Never post the proxy login anywhere. It stays private between you and your manager.

2 Create the account (4 platforms)

Spread the articles across all four so the profile looks natural. Steps are the same idea on each: sign up → set the author name to the article's persona → create a blog/subdomain.

Medium — medium.com

  1. Go to medium.com → Get started → sign up with the fresh email.
  2. Set your name to the article's persona name; skip the topics step.
  3. Click Write (top right) to open the editor.

Blogger — blogger.com

  1. Go to blogger.com → sign in with the fresh Gmail → Create New Blog.
  2. Title = a plain topic name (e.g. "Wellness Notes"); pick any available .blogspot.com address.
  3. Click New Post.

WordPress.com — wordpress.com

  1. Go to wordpress.com → Start your website → sign up with the fresh email.
  2. Choose the free plan; pick any available .wordpress.com subdomain.
  3. Open the dashboard → Posts → Add New.

Tumblr — tumblr.com

  1. Go to tumblr.com → sign up with the fresh email → pick a blog name.
  2. Click the pencil/Text post button.

3 Publish the article

  1. Copy the article Title (button on the article card) → paste into the post title.
  2. Copy the article Body → paste into the post body. It keeps its headings and paragraphs.
  3. Insert the links: for each link listed on the article card — highlight the exact anchor text in the body, click the editor's link button (🔗), paste the matching URL, press Enter. (Turn OFF "open in new tab" if asked.)
  4. Add 3–5 tags: weight loss, supplements, reviews, health.
  5. Click Publish. Open the live post and check every link works and points to the right osanix.shop page.
